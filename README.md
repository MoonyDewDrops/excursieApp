# Excursie App Berlijn

Een responsive excursie-app over Berlijn, gemaakt met React, Vite en Express.

## Functionaliteiten

* Homepagina
* Overzicht van bezienswaardigheden
* Overzicht van activiteiten
* Praktische informatie
* Zoekfunctie
* Favorieten opslaan
* Weerwidget 
* CMS met:

  * Inloggen
  * Uitloggen
  * Beveiligde beheeromgeving
  * Toevoegen, wijzigen en verwijderen van gegevens (CRUD)

## Gebruikte technieken

### Frontend

* React
* Vite
* React Router

### Backend

* Express
* MySQL
* bcrypt
* CORS
* dotenv
